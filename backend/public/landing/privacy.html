StayLabel プライバシーポリシー

StayLabel（以下「当社」といいます。）は、当社が提供する宿泊予約管理サービス（以下「本サービス」といいます。）における、利用者の個人情報を含むプライバシー情報の取扱いについて、以下のとおりプライバシーポリシーを定めます。

第1条（取得する情報）

当社は、本サービスの利用にあたり、以下の情報を取得する場合があります。

  • 氏名、メールアドレス、電話番号などのお客様情報
  • 予約内容（チェックイン・チェックアウト日、宿泊施設名など）
  • お問い合わせ時にいただく情報
  • Gmail API 等を通じて取得するメール送受信データ（送信者・受信者、件名、本文、日時など）

第2条（利用目的）

当社が取得した情報は、以下の目的のために利用します。

  • 宿泊予約や問い合わせ対応など、本サービスの提供のため
  • 本サービスに関する連絡・通知のため
  • 不正利用やトラブル防止のため
  • 本サービスの改善、新機能開発のため
  • 法令に基づき必要とされる場合

第3条（第三者提供）

当社は、次の場合を除き、取得した個人情報を第三者に提供することはありません。

  • ご本人の同意がある場合
  • 法令に基づく場合
  • 生命・身体・財産の保護が必要で同意を得ることが困難な場合
  • 業務委託先に、本サービス運営に必要な範囲で委託する場合

第4条（安全管理）

当社は、取得した個人情報を適切に管理し、不正アクセス、漏洩、改ざん、滅失を防止するために必要な措置を講じます。

第5条（開示・訂正・削除）

ご本人から自己の個人情報について開示、訂正、利用停止または削除の請求があった場合には、法令に基づき速やかに対応します。

第6条（プライバシーポリシーの変更）

本ポリシーの内容は、法令の変更やサービス内容の改善等に応じて、事前の予告なく変更されることがあります。変更後の内容は本ウェブサイトに掲載した時点で効力を生じます。

第7条（お問い合わせ窓口）

本ポリシーに関するお問い合わせは、以下の窓口までお願いします。
